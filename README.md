# Playnite Browser

This is a Playnite extension to let you add websites as games!

## Installation

To install this extension, go to [the Playnite Extension store](https://playnite.link/addons.html#PlayniteBrowser_3c58c307-b26e-4569-855a-e19afa4a3b2f) and click download!
